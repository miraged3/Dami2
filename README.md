# 基于Mirai2.0开发的船新大米

## 介绍

无情的阴阳师抽卡bot

所有功能都在设计与开发中，将开发的实时进度上传至Github~~刷提交记录~~

## 使用

### 1.环境配置

Java 11+

### 2.配置config.properties配置文件，内容如下

| 配置 | 说明 |
| ---- | ----|
| qq.account= | 你的QQ账号 |
| qq.password= | 该账号的密码 |
| qq.admin= | 管理员的QQ号 |
| curse= | 脏话关键字，用英文逗号分割 |
| daily.url=http://open.iciba.com/dsapi/ | 金山词霸每日一句链接，默认不用修改

### 3.启动大米

Windows：StartDami.bat

Linux ：./StartDami.sh
